Folders

The rAPPNUWSAnalyzer incorporates the following folder structure :
  • ODB
  • rAPPNUWSData
    • GeantTrack
    • DCGeantStep
    • CSMBTGeantStep
    • CSMTTGeantStep
    • MCTrack
  • MC
    • MCRunHeader
    • DCHHit
    • CSMBTHit
    • CSMTTHit
  • RecoData
    • RecoTracks
  • AnalyzerConfigParameters
The bold folders are data folders. The others are only used to structure the frame work.

Trees

The rAPPNUWSAnalyzer incorporates the following trees and branches. In brackets are the folders, which are filled to the branch.
  • MCData Simulation specific data. It a simple conversion from Geant4 stepper
    • MCTracks (GeantTrack)
    • MCDCStep (DCGeantStep)
    • MCCSMBTStep (CSMBTGeantStep)
    • MCCSMTTStep (CSMTTGeantStep)
  • MCHits Container for MC Hits
    • MCTracks (GeantTrack)
    • DCHits (DCHHit)
    • CSMBTHits (CSMBTHit)
    • CSMTTHits (CSMTTHit)
  • RecoData Container for RecoData
    • RecoTracks (RecoTracks)
